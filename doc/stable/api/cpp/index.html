Logging levels¶

enum LIEF::LOGGING_LEVEL¶

Hierarchical logging level

From a given level set, all levels below this level are enabled

For example, if LOG_FATAL is enabled then LOG_ERROR, LOG_WARNING are also enabled

Values:

LOG_GLOBAL = 1¶
LOG_TRACE = 2¶
LOG_DEBUG = 4¶
LOG_FATAL = 8¶
LOG_ERROR = 16¶
LOG_WARNING = 32¶
LOG_INFO = 64¶
